# HR-Game-Hackathon
Phaser Starter Repo w/Express Server for Hack Reactor's Game Hackathon

## Getting Started

From within the root directory:
```sh
* npm install
```
Running the Server
```sh
* nodemon server.js
```
Controls in the Demo are the up/left/right arrow keys
